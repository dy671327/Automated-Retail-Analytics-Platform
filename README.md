# 📊 Automated Retail Analytics Platform

An end-to-end Retail Analytics solution built in Microsoft Excel using Power Query, Power Pivot, DAX and the Excel Data Model.

This project demonstrates data cleaning, ETL, data modelling, KPI reporting and interactive dashboard development using a real-world e-commerce dataset.

---

# 🚀 Project Overview

The objective of this project is to transform raw e-commerce data into interactive dashboards that help business users monitor sales performance, customer insights and key business metrics.

---

# 🛠 Tools & Technologies

- Microsoft Excel
- Power Query
- Power Pivot
- DAX
- Pivot Tables
- Pivot Charts
- Excel Data Model

---

# 📈 Dashboard Features

### Executive Dashboard

- Total Sales
- Gross Revenue
- Total Orders
- Total Customers
- Monthly Sales Trend
- Top Product Categories
- Top States by Sales
- Average Review Score
- Interactive Slicers

### Customer Insights

- Total Customers
- Average Revenue per Customer
- Orders per Customer
- Interactive Customer Filters
